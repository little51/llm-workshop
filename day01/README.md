# 第1天 大语言模型导论

## 一、提示词技巧

```powershell
1、一般技巧
（1）角色定义（可选）
（2）任务指令（核心）
（3）补充细节（关键）
（4）输出格式（可选）
2、适用场景
3、专业领域
4、限定生成范围
5、对输出格式有要求
```

## 二、提示词样例

```powershell
你是一位专业的中英文校对编辑

请对以下文本进行严格校对:
{原文}

要求:
1.语法修正(时态、主谓一致、冠词等)
2.标点符号规范化(中英文标点区分、引号使用等)3.逻辑连贯性检查(段落衔接、转折词合理性)4.术语一致性(如"A!"和"人工智能"统一为后者)5.避免冗余表达(删除重复或无意义的修饰词)

请输出校对后的原文，并用以下格式列出校对结果
1.原句修改后修改原因
输出格式(可选)
2.原句修改后修改原因

```
